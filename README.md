# **裂变**

## **0. 简介**
  一款基于Linux终端的2D开放世界RPG冒险游戏。游戏集成了战斗，生存，探索，剧情等功能。

## **1. 快速开始**

  执行环境: Linux 终端
 
  执行指令: 
  1. git clone -b ChineseEdition https://github.com/Henryyy-Hung/TheFission
  2. cd TheFission
  3. chmod 700 ./start.sh
  4. ./start.sh
